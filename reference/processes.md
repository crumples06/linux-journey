# Commands
## ps command
Reports a snapshot of the current processes.    

## htop command
Real time update of processes and is interactive 

## kill command
`kill process_id`. Sends a signal to the process asking it to clean up open files, save state and exit gracefully.
`kill -9 process_id`. Kills the process 

## jobs command
Lists all the background jobs in the current shell session.

## pgrep command
Gives the pid of the specified process.
